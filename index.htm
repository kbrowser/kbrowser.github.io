    • kScenes is a handy viewer for pictures in PNG, GIF, HEIC, JPEG, WEBP, BMP, XCF, TIFF, DCM and DDS formats. These formats can be converted to the format PNG, JPEG or GIF. kScenes supports resizing and clipping a picture.

    kScenes for Windows

    The app only runs on 64-bit systems. Windows 11 is recommended.
