    • Latest updates

    Version 1.8

    kScenes has been redesigned using PySide6 which is Qt module for Python.

    Version 1.7

    kScenes now can show detailed Exif of images.

    Version 1.6

    1. Since this version, kScenes for Windows includes both of Pillow and ImageMagick for handling images.

    2. Since this version, kScenes for Linux includes Pillow for handling images. ImageMagick is a recommended option.

    3. Viewing performance is much improved because of using Pillow. ImageMagick is necessary for handling HEIC, XCF and DCM images.

    Version 1.5

    1. Now you can resize a picture with a given width or a given height.

    2. Clipping feature is now in clip mode, in order to avoid the conflict of left mouse button.

    3. Viewer Mode is added. It only works when you open a picture through system menu.

    Version 1.4

    1. Now the app can show the subdirectories in default pictures folder.

    2. Right-click menus have an alternative way through pressing left mouse button.

    3. In the album interface you may press left mouse button and slide mouse up or down for scrolling the album.

    Version 1.3

    Now kScenes has a dark UI, and the experience for a touch screen is optimized. And since this version, Necessary components of ImageMagick are included for out-of-the-box use.

    Version 1.2

    1. Now images with different width can be shown compactly.

    2. The pop-up viewer can show a picture with full size, half size or one-third size.

    3. Using the pop-up viewer, you can convert any picture to PNG, JPEG or GIF format.

    4. You can clip an area of a picture and save it to PNG, JPEG or GIF format.
